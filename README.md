# 🛍️ Customer Shopping Behavior Analysis

A comprehensive data analytics project that explores customer shopping behavior using **Python, PostgreSQL, and Power BI**. This project focuses on cleaning and transforming retail transaction data, performing SQL-based business analysis, and creating an interactive Power BI dashboard to uncover customer purchasing trends and actionable business insights.

---

## 📌 Project Overview

Businesses generate massive amounts of transactional data every day. Analyzing this data helps organizations understand customer behavior, optimize marketing strategies, improve customer retention, and drive business growth.

This project analyzes **3,900 customer purchase records** to answer key business questions related to customer demographics, spending habits, product performance, subscription behavior, discounts, and sales trends.

### Workflow

**Dataset → Data Cleaning (Python) → SQL Analysis (PostgreSQL) → Dashboard (Power BI) → Business Insights**

## 📈 Project Workflow

### 1. Data Cleaning & Preprocessing (Python)

The dataset was cleaned and transformed using Python.

Tasks performed:

- Loaded the dataset using Pandas
- Explored data using `info()` and `describe()`
- Handled missing values
- Filled missing review ratings using category-wise median
- Renamed columns using snake_case
- Created Age Group feature
- Created Purchase Frequency feature
- Removed redundant columns
- Exported cleaned data to PostgreSQL

---

### 2. Business Analysis (PostgreSQL)

Business questions answered using SQL:

- Revenue generated by Gender
- High Spending Discount Users
- Top 5 Highest Rated Products
- Shipping Type Comparison
- Subscribers vs Non-Subscribers
- Discount Dependent Products
- Customer Segmentation
- Top 3 Products per Category
- Repeat Buyers & Subscription Analysis
- Revenue by Age Group

---

### 3. Dashboard Development (Power BI)

An interactive dashboard was developed to visualize business insights.

Dashboard includes:

- KPI Cards
- Revenue by Category
- Sales by Category
- Subscription Distribution
- Revenue by Age Group
- Sales by Age Group
- Interactive Filters

---

## 📊 Key Insights

- Young adults contributed the highest revenue.
- Loyal customers formed the largest customer segment.
- Express shipping customers spent slightly more on average.
- Some products were highly dependent on discounts.
- Subscribers and non-subscribers showed similar average spending patterns.
- Top-rated products can be prioritized in future marketing campaigns.

---

## 💡 Business Recommendations

- Increase subscription adoption through exclusive offers.
- Introduce customer loyalty and reward programs.
- Promote top-rated products in marketing campaigns.
- Optimize discount strategies to improve profitability.
- Focus marketing efforts on high-revenue customer segments.
